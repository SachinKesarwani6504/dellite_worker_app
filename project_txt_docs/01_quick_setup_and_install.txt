QUICK SETUP + INSTALL (COPY/PASTE)

1) CREATE APP (EXPO + TS)
npx create-expo-app dellite_customer_app -t expo-template-blank-typescript
cd dellite_customer_app

2) INSTALL CORE LIBRARIES (MATCH CURRENT WORKER APP)
npm install @expo/vector-icons @react-navigation/bottom-tabs @react-navigation/native @react-navigation/native-stack axios expo-font expo-linear-gradient expo-secure-store expo-status-bar lottie-react-native nativewind react-native-gesture-handler react-native-keyboard-aware-scroll-view react-native-keychain react-native-modal react-native-paper react-native-reanimated react-native-safe-area-context react-native-screens react-native-svg react-native-toast-message react-native-worklets tailwindcss

3) INSTALL DEV LIBRARIES
npm install -D @types/react babel-plugin-module-resolver babel-preset-expo typescript

4) BABEL CONFIG (babel.config.js)
- Use presets: ['babel-preset-expo', 'nativewind/babel']
- Use module-resolver alias: '@' -> './src'
- Keep 'react-native-reanimated/plugin' in plugins.

5) TYPESCRIPT CONFIG (tsconfig.json)
- strict: true
- types: ["nativewind/types"]
- baseUrl: "."
- paths: { "@/*": ["src/*"] }

6) TAILWIND CONFIG (tailwind.config.js)
- presets: [require('nativewind/preset')]
- darkMode: 'media'
- content: ['./App.{js,jsx,ts,tsx}', './src/**/*.{js,jsx,ts,tsx}']
- extend colors:
  brandOrange: #FF7A00
  brandRed: #FF3D5A
  brandYellow: #FFC53D
  brandCream: #FFF7E8
  brandText: #2D1B00
  brandBlack: #0F0F10

7) GLOBAL CSS (global.css)
@tailwind base;
@tailwind components;
@tailwind utilities;

8) RUN APP
npm run start
npm run android
npm run ios
